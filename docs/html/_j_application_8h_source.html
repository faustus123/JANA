83 /// JANA application class (singleton).
84 ///
85 /// The JApplication class serves as a central access point for getting to most things
86 /// in the JANA application. It owns the JThreadManager, JParameterManager, etc.
87 /// It is also responsible for making sure all of the plugins are attached and other
88 /// user specified configurations for the run are implemented before starting the processing
89 /// of the data. User code (e.g. plugins) will generally register things like event sources
90 /// and processors with the JApplication so they can be called up later at the appropriate time.
91 //////////////////////////////////////////////////////////////////////////////////////////////////
