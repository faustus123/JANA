void JEventSourceManager::AddEventSource ( const std::string &  source_name)

Add the named event source to the list of event sources to read from. Usually, these are taken from the command line, but this allows cutom plugins to add a source programmatically. This will be added to the end of the current list of source names (i.e. after any entered on the command line.)
